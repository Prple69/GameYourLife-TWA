<task type="checkpoint:human-verify" gate="blocking">
  <what-built>
Complete Phase 5 Shop & Inventory system:
- Shop catalog (GET /api/shop) with 24 seed items, 4 filter tabs, retro card layout, confirm-buy flow
- Inventory page with 4 sections: active effects, boosters, skins, potions — with activate/equip/use buttons
- Header active boost countdown timer row (client-side ticking)
- Quest add button {count}/5 cap with disable at 5
- AvatarSelector premium skin lock overlays with shop navigation
- Atomic purchase/activation with idempotency keys
  </what-built>
  <how-to-verify>
1. **Shop Catalog:**
   - Navigate to `/app/shop`
   - Verify items load (should see ~24 items)
   - Click each filter tab: «Бусты», «Скины», «Зелья», «Все» — verify correct items shown
   - Verify retro card layout: black glass cards, gold price button on right, icon on left

2. **Insufficient gold UI:**
   - Note your current gold amount
   - Find a shop item that costs more than your gold
   - Verify: price button is dimmed (opacity-50), price text is red
   - Click it → verify toast «Не хватает gold (есть: X, нужно: Y)»

3. **Buy item:**
   - Find a cheap item (e.g., «Малый эликсир опыта» — 100 gold)
   - Click Buy → verify ConfirmModal appears with correct name and price
   - Click Cancel → verify modal closes, no purchase
   - Click Buy again → confirm → verify toast «Куплено: [name] ✓»
   - Verify gold decreased by item price (check header)
   - Navigate to `/app/inventory` → verify item appears in Бусты section

4. **Activate boost:**
   - On InventoryPage, click «Активировать» on an owned boost
   - Verify toast «[name] активирован!»
   - Verify boost appears in Header with mm:ss countdown (should be ticking)
   - Wait 3 seconds → verify timer decremented
   - Try activating same boost type again → verify button is disabled

5. **Use healing potion (if HP < max HP):**
   - If HP is full, fail a quest first to take damage
   - On InventoryPage Зелья section, click «Использовать»
   - Verify toast «+N HP восстановлено»
   - Verify HP in header increased

6. **Buy and equip skin:**
   - Buy any skin (e.g., «Маг» — 500 gold, if you have enough)
   - Go to Inventory → Скины section → verify skin appears
   - Click «Экипировать» → verify selected_avatar highlighting changes
   - Navigate to Character page → verify avatar changed

7. **AvatarSelector locks:**
   - Open AvatarSelector (e.g., via Character page or Profile modal)
   - Verify unowned premium skins show grayscale + lock icon + price
   - Click a locked skin → verify navigates to `/app/shop` (with or without filter param)

8. **Quest slot cap:**
   - Go to `/app/quests`
   - Verify «+ Добавить» button shows count subtitle (e.g., «2/5 квестов»)
   - If fewer than 5 active quests: button should be enabled
   - If you can add quests to reach 5: verify button becomes disabled at exactly 5 active

9. **Quest completion with boost:**
   - With an active XP boost, complete a quest
   - Verify completion toast/popup shows boost attribution (e.g., «⚡ XP-буст ×1.5»)
   - Verify XP gained is multiplied vs. what it would normally be
  </how-to-verify>
  <resume-signal>Type "approved" if all checks pass, or describe any issues found (e.g., "Step 3 failed: modal doesn't close after confirm")</resume-signal>
