**3. effective_max_hp(user, now: datetime) -> int**
Returns `user.max_hp + hp_max_bonus` if hp_max boost active, else `user.max_hp`.
Test cases:
- `test_no_hp_boost`: `user.max_hp=100, active_hp_max_bonus=None` → 100
- `test_hp_boost_active`: `user.max_hp=100, active_hp_max_bonus=50, expires_at=now+60s` → 150
- `test_hp_boost_expired`: `expires_at=now-1s` → 100

Implementation guideline (GREEN):
```python
BOOST_MULT_TYPES = ("xp", "gold", "strength_xp", "wisdom_xp", "endurance_xp", "charisma_xp")

def effective_multipliers(user, now) -> dict:
    result = {}
    for btype in BOOST_MULT_TYPES:
        mult = getattr(user, f"active_{btype}_mult", None)
        exp  = getattr(user, f"active_{btype}_expires_at", None)
        result[btype] = (mult if (mult is not None and exp is not None and exp > now) else 1.0)
    return result

def effective_max_hp(user, now) -> int:
    bonus = getattr(user, "active_hp_max_bonus", None)
    exp   = getattr(user, "active_hp_max_expires_at", None)
    if bonus is not None and exp is not None and exp > now:
        return user.max_hp + bonus
    return user.max_hp
```
