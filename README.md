# GameYourLife 🎮

**GameYourLife** — это Telegram Web App (TWA), которое превращает рутинные задачи и достижение целей в захватывающую игру. Отслеживайте свои привычки, выполняйте квесты и прокачивайте своего персонажа в реальной жизни через интерфейс Telegram.

## ✨ Особенности
- **Геймификация:** Получайте опыт (XP) и уровни за выполнение задач.
- **Интеграция с Telegram:** Работает прямо внутри мессенджера (Mini App).
- **Адаптивный дизайн:** Оптимизировано для мобильных устройств.
- **Трекер привычек:** Визуализация прогресса и система напоминаний.

## 🚀 Технологии
- **Frontend:** React.js / Vite (или HTML5/JS)
- **Стили:** CSS Modules / Tailwind CSS
- **Telegram SDK:** `@twa-dev/sdk`
- **Backend (опционально):** Node.js / Firebase
